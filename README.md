## Overview
Manul is an adapter skeleton, provides ability to switch 'http to socket' or 'socket to http'.

## Required Dependencies
* cJSON

## Building
* cd manul
* ./rebuild

## Run
1. copy "mnl", "conf/mnl.lua" to your project dir
2. config "manul.lua"
3. run with "./mnl"
